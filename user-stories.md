User Stories

## Authentication
- As a user ISBAT register an account (Sign Up)
- As a user ISBAT login to my account
- As a user ISBAT logout from my account

## Profile System
- As a user ISBAT view my profile
- As a user ISBAT edit my profile (bio, profile picture)
- As a user ISBAT view other users’ profiles

## Posts (Core Feature)
- As a user ISBAT create a post (image and caption)
- As a user ISBAT view all posts (basic feed)
- As a user ISBAT view a single post
- As a user ISBAT delete my own post

## Likes
- As a user ISBAT like a post
- As a user ISBAT unlike a post
- As a user ISBAT view number of likes on a post

## Comments
- As a user ISBAT add a comment on a post
- As a user ISBAT delete my own comment
- As a user ISBAT view comments on a post

## Follow System
- As a user ISBAT follow another user
- As a user ISBAT unfollow a user
- As a user ISBAT view list of followers and following

## Search and Error Handling
- As a user ISBAT search for users by username
- As a user ISBAT see an error message for invalid login
- As a user ISBAT see a 404 page for invalid URLs
